# Tuya SNZB-02 Temperature and Humidity Sensor

Tuya SNZB-02 is a Hubitat Zigbee driver area for the SNZB-02 temperature and humidity sensor.

## Files in this area

- `Drivers/TuyaSnzb02TemperatureHumidityDriver.groovy` — Zigbee driver
- `INSTALLATION.md` — installation and usage
- `Specification.md` — detailed implementation specification

## Summary

- reads temperature 
- reads humidity
- reads battery
- supports refresh and configure operations
- includes a temperature offset preference
